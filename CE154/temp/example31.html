Important Points

CSS Position

Can position element in four ways:

Static: Positioned according to normal flow (see above). Not affected by top, bottom, left, and right properties.

Fixed: Positioned relative to the browser window. Will not move even if content of window is scrolled.

NB: Removed from normal flow and independent of it. Can overlap with other elements.

Relative: Positioned relative to its position in normal flow.

NB: Space it takes in normal flow remains the same. Can overlap with other elements.

Absolute: Positioned according to first parent element which is not static (i.e. normal flow). Often this is html element, i.e. whole document.
